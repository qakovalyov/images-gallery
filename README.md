# Rails Images-gallery

## Installation:

``` install rvm
    install ruby, '->2.1.5'
    rvm use ruby-2.1.5@images-gallery
    bundle
    run postgresql server
    rake db:drop db:create db:migrate db:seed
    rails s
```
